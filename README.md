## **🛒 Multi-Store Tech Scraper (Alkosto Edition)**

 Este proyecto consiste en un web scraper avanzado desarrollado con Python y Scrapy, diseñado para extraer información de productos tecnológicos (Computadores, Celulares y Tablets) de la tienda Alkosto. Utiliza Playwright para gestionar contenido dinámico y carga infinita, almacenando los resultados en una base de datos NoSQL (MongoDB Atlas).

## 🚀 Características Navegación Dinámica:

 Uso de scrapy-playwright para interactuar con botones de "Cargar más" y gestionar el renderizado de JavaScript.

Bucle Multi-Categoría: Extracción automatizada de múltiples secciones del sitio web en una sola ejecución.

Procesamiento de Datos: * Normalización de precios a formato numérico.

Categorización automática basada en palabras clave.

Extracción inteligente de Marcas mediante selectores CSS y lógica de respaldo (Fallback).

Persistencia: Integración directa con MongoDB Atlas para almacenamiento en la nube.

## 🛠️ Tecnologías Utilizadas Lenguaje: Python 3.x

Framework de Scraping: Scrapy

Automatización de Navegador: Playwright

Base de Datos: MongoDB Atlas

Gestión de Eventos: Asyncio (Configurado para compatibilidad con Windows)
